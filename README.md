## Сайт prodigyday.ru  

Сайт является базой данных событий по дням. Выводит на главной странице событие текущего дня в разные годы. Можно смотреть архив всех событий по годам и всех событий в определенный день разных лет.  

Python  
PostgreSQL  
Django  
Bootstrap
